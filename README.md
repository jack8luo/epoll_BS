# epoll_BS
linux下的一个基于epoll多路IO转发的本地文件传输服务器。没有做压力测试，没有使用线程池。
# 使用
 ## 前提：在linux系统下，安装gcc编译环境，makefile环境。
 # 步骤
 1、make 
 
 2、./server 9876 /home/youname/dir/
 
 3、打开浏览器，目测edge能行
